Meus Projetos

Connect 4

Este projeto implementa o clássico jogo Connect 4 com a opção de jogar contra uma IA usando dois algoritmos de tomada de decisão: Minimax e Alpha-Beta Pruning. O jogo permite escolher entre esses algoritmos e configurar a profundidade de pesquisa (ply) para testar o desempenho de cada um.

Criptografia

Este projeto foi desenvolvido utilizando o algoritmo AES (Advanced Encryption Standard) para criptografar e descriptografar mensagens de texto. O sistema fornece uma interface simples onde os usuários podem inserir uma mensagem, criptografá-la usando uma chave secreta definida pelo usuário e posteriormente descriptografá-la usando a mesma chave.

Gerador de Senhas

Este projeto é um aplicativo gerador de senha desenvolvido em Android Studio com Kotlin. O aplicativo permite que os usuários criem senhas seguras e personalizáveis, escolhendo opções específicas, como letras maiúsculas e minúsculas, números, caracteres especiais e comprimento da senha.
